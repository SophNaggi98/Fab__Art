How do I return my order?

If you are not satisfied with your order, you have 100 days to send the unwanted goods back to me - completely free of charge.
You can now easily return parcels with a QR code on your phone. Register your return online below and tell me which items you would like to return.
Once you've created the return, you'll receive a code that you show on your mobile phone when you drop off the parcel.

How to create a return

1. Click the ‘Return goods’ button below.
2. Select the items you want to return and choose a reason.
3. Select your preferred return method - return code or return label.
4. You can now save the code on your mobile and show it when you hand in the parcel. Alternatively, you can print out a return label and attach it to the parcel yourself.
5. Pack your return and drop it off at a collection point or post office. If you use a return code, show it to the employee and they will print a return label for you.
6. Don't forget to ask for a receipt for the return. And then you're done!
